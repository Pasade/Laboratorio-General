🏁 Introducción a las Matrices: El Garaje de F1Hasta ahora, un array era como una sola fila de cajas en el taller. Pero un equipo de F1 tiene varios coches y quiere medir sus datos a la vez. Para eso usamos una Matriz, que no es más que una tabla con filas y columnas.Si queremos guardar los tiempos de 2 coches durante 3 vueltas, creamos una cuadrícula de 2x3:Vuelta 1 (Columna 0)Vuelta 2 (Columna 1)Vuelta 3 (Columna 2)Coche 1 (Fila 0)74.273.873.1Coche 2 (Fila 1)75.074.173.5En Java se declara pasando dos pares de corchetes [][] (el primero para las filas y el segundo para las columnas):Javadouble[][] tiemposEquipo = {
    {74.2, 73.8, 73.1}, // Fila 0 (Coche 1)
    {75.0, 74.1, 73.5}  // Fila 1 (Coche 2)
};
Para acceder a una caja exacta, necesitas darle las dos coordenadas: tiemposEquipo[fila][columna].
