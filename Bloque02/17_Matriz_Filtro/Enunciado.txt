Reto 17: Matriz Filtro (Control de Neumáticos Críticos) 🏎️⚠️
El equipo de ingenieros te ha pedido una herramienta para monitorizar el desgaste de los neumáticos de 3 coches durante 4 vueltas de simulación (una matriz de 3x4). Quieren saber exactamente en qué momentos el desgaste superó un umbral peligroso (80%), ya que a partir de ahí hay riesgo de reventón.

📋 El Objetivo
Recorrer una matriz completa usando bucles anidados (for dentro de for) y aplicar un filtro condicional para reportar solo las posiciones que superan un valor límite.

⚙️ Lógica que debes programar
En tu método main, declara la siguiente matriz de tipo int[][] que contiene el desgaste de 3 coches a lo largo de 4 vueltas:

Java
int[][] telemetriaNeumaticos = {
    {20, 45, 65, 82},  // Fila 0: Coche 1
    {15, 35, 78, 90},  // Fila 1: Coche 2
    {30, 60, 85, 95}   // Fila 2: Coche 3
};
Imprime un encabezado limpio en la consola: "--- ALERTA DE DESGASTE CRÍTICO (>80%) ---".

Crea dos bucles for anidados para revisar cada casilla de la tabla:

El bucle exterior recorrerá las filas (int c = 0; c < 3; c++).

El bucle interior recorrerá las columnas/vueltas (int v = 0; v < 4; v++).

El Filtro (if): Dentro del bucle interior, evalúa si el valor de la casilla actual es mayor que 80.

Si es verdadero, imprime un reporte con printf indicando qué coche y en qué vuelta se ha detectado el peligro.

Recuerda la regla de oro: Súmale 1 a las variables al imprimirlas para que de cara al usuario se vea de forma natural (Coche 1, Vuelta 4...) y no con los índices de programación (Coche 0, Vuelta 3).

📂 Tu rutina en la terminal:
Asegúrate de estar en tu carpeta del bloque 2: cd .. (si estás dentro de otra).

Entra en la carpeta del ejercicio (o créala si no existe): cd 17_Matriz_Filtro

Si está vacía, crea tu archivo: touch Main.java

Móntalo, lánzale el javac Main.java y pásame la captura para ver cómo ese filtro caza los desgastes críticos en la pista. ¡A por ello, que lo tienes dominado!
